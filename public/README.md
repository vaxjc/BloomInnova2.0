# Public Assets

Esta carpeta contiene los activos públicos del proyecto Bloom Innova.

## Contenido

- **logo.svg** - Logo principal de Bloom Innova (200x200px)
- **favicon.ico.svg** - Favicon del sitio web
- Otros activos estáticos que se sirven públicamente

## Uso

Los archivos en esta carpeta se sirven automáticamente en la raíz del sitio web:
- `/public/logo.svg` → `https://bloominnova.com/logo.svg`
- `/public/favicon.ico.svg` → `https://bloominnova.com/favicon.ico.svg`

## Notas

- Todos los archivos SVG están optimizados para web
- Los logos mantienen la paleta de colores de Bloom Innova
- Asegúrate de comprimir las imágenes antes de hacer deploy
